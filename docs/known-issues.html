Chapter 18 Known issues

Alpha known issues:

  • By default users are not allowed to create operators unless explicitly given permission by a Tercen administrator
  • Cannot perform an operator on multi pairwise projections
  • If your local computer’s date and clock time is not the same as the Tercen server then you may get delays and errors
  • Cannot run python scripts

Beta known issues: The beta version of Tercen is currently being tested.
